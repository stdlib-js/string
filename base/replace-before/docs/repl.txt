
{{alias}}( str, search, replacement )
    Replaces the substring before the first occurrence of a specified search
    string. 

    Parameters
    ----------
    str: string
        Input string.

    search: string
        Search string.
    
    replacement: string
        Replacement string.

    Returns
    -------
    out: string
        Output string.

    Examples
    --------
    > var str = 'beep boop';
    > var replacement = 'foo';
    > var out = {{alias}}( str, ' ', replacement )
    'foo boop'
    > out = {{alias}}( str, 'o', replacement )
    'foooop'
