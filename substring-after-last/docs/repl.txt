{{alias}}( str, search )
    Returns the part of a string after the last occurrence of a specified
    substring.

    Parameters
    ----------
    str: string
        Input string.

    search: string
        Search value.

    Returns
    -------
    out: string
        Substring.

    Examples
    --------
    > var out = {{alias}}( 'beep boop beep baz', 'beep' )
    ' baz'

    > out = {{alias}}( 'Hello World!', 'Hello ' )
    'World!'
