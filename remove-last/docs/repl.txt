
{{alias}}( str )
    Removes the last character of a `string`.

    Parameters
    ----------
    str: string
        Input string.

    Returns
    -------
    out: string
        Updated string.

    Examples
    --------
    > var out = {{alias}}( 'beep' )
    'bee'
    > out = {{alias}}( 'Boop' )
    'Boo'
